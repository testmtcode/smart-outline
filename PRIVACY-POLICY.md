# SmartOutline 隐私政策

**生效日期：2026年4月7日**

## 概述

SmartOutline（"我们"、"本插件"）是一款浏览器扩展，为用户提供网页目录大纲功能。我们高度重视用户隐私，本隐私政策说明了我们如何处理用户数据。

## 数据收集

**SmartOutline 不收集、存储或传输任何用户个人信息。**

具体说明：
- ❌ 不收集浏览历史
- ❌ 不收集网页内容
- ❌ 不收集个人身份信息
- ❌ 不收集使用数据
- ❌ 不上传任何数据到服务器

## 权限说明

本插件需要以下权限，仅用于实现核心功能：

| 权限 | 用途 | 数据去向 |
|------|------|----------|
| `activeTab` | 读取当前页面标题结构生成大纲 | 仅在本地处理，不上传 |

## 数据处理

所有数据处理均在用户浏览器本地完成：
- 页面标题分析在本地进行
- 大纲数据存储在浏览器内存中
- 不涉及任何服务器通信

## 第三方服务

本插件不集成任何第三方分析、广告或追踪服务。

## 数据安全

由于本插件不收集或传输任何用户数据，不存在数据泄露风险。

## 用户权利

用户可以随时：
- 卸载插件，所有相关数据将立即清除
- 查看插件源代码（开源）
- 控制插件权限（通过浏览器设置）

## 变更通知

如本隐私政策发生变更，我们将：
- 在插件更新时提示用户
- 在此页面更新生效日期

如有任何隐私相关问题，请通过以下方式联系：

# SmartOutline Privacy Policy

**Effective Date: April 7, 2026**

## Overview

SmartOutline ("we", "this extension") is a browser extension that provides webpage outline functionality. We value user privacy and this policy explains how we handle user data.

## Data Collection

**SmartOutline does NOT collect, store, or transmit any user personal information.**

Specifically:
- ❌ No browsing history collected
- ❌ No webpage content collected
- ❌ No personally identifiable information collected
- ❌ No usage data collected
- ❌ No data uploaded to any server

## Permissions

This extension requires the following permissions for core functionality only:

| Permission | Purpose | Data Destination |
|------------|---------|------------------|
| `activeTab` | Read page heading structure to generate outline | Local processing only, no upload |

## Data Processing

All data processing happens locally in the user's browser:
- Page heading analysis is done locally
- Outline data is stored in browser memory
- No server communication involved

## Third-Party Services

This extension does not integrate with any third-party analytics, advertising, or tracking services.

## Data Security

Since this extension does not collect or transmit any user data, there is no risk of data breach.

## User Rights

Users can at any time:
- Uninstall the extension, all related data will be immediately cleared
- View the source code (open source)
- Control extension permissions (via browser settings)

## Changes

If this privacy policy changes, we will:
- Notify users when updating the extension
- Update the effective date on this page

## Contact

For privacy-related questions, please contact:
- GitHub Issues: https://github.com/yourname/smart-outline/issues
